## üí° Features

* **Basic Operations:** Supports addition (`+`), subtraction (`-`), multiplication (`*`), and division (`/`).
* **Display:** A large, visible display screen for inputs and results.
* **Clear All Button (`AC`):** Clears the entire current input.
* **Delete/Backspace Button (`DE`):** Deletes the last character entered.
* **Decimal Support:** Includes the decimal point button (`.`).
* **Double Zero (`00`):** Button for fast input of zeros.
